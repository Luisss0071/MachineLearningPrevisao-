Score de Crédito dos Clientes

Este projeto tem como objetivo definir o score de crédito de novos clientes em um banco. 
O sistema analisa todos os clientes do banco e, com base nessa análise, cria um modelo que 
consiga ler as informações do cliente e dizer automaticamente o score de 
crédito dele: Ruim, Ok, Bom. Isso será necessário para poder prever a nota de crédito de novos clientes.
